# KidsCare

## Deskripsi

KidsCare adalah aplikasi layanan kesehatan anak berbasis web yang membantu orang tua, dokter, dan admin dalam mengelola layanan kesehatan anak secara digital.

## Fitur

### Orang Tua

* Login dan Logout
* Mengelola profil
* Menambah data anak
* Mengedit data anak
* Menghapus data anak
* Membuat janji temu dengan dokter
* Melihat riwayat janji temu
* Melihat rekam medis anak

### Dokter

* Login dan Logout
* Melihat daftar pasien
* Mengelola jadwal pemeriksaan
* Menyetujui atau menolak janji temu
* Mengisi rekam medis pasien
* Mengelola profil dokter

### Admin

* Login dan Logout
* Dashboard statistik
* Mengelola data dokter
* Mengelola janji temu
* Monitoring data sistem

### Frontend

* React
* Vite
* React Router DOM
* Axios
* CSS

### Backend

* Node.js
* Express.js
* JWT Authentication
* Cookie Authentication
* REST API
* CORS

### Database

* PostgreSQL
